HTML

  • HTML elements work together to build the structure of a webpage
  • The head element contains information about the webpage
  • The body element represents the visible content shown to the user
  • To add content to pages use text, image, list and header elements
  • To divide and describe a page use meta-related, content division and semantic elements
  • HTML plays in making websites readable and easy to use

CSS

  • CSS determines the styles of a webpage in order to make a website visually appealing
  • Declarations contain the CSS property we want to apply and the value of the property
  • EG color: pink; includes the CSS property colour and the value for that property pink
  • A margin property indicates how much space we want around the outside of an element
  • A padding property adds space around the content inside an element
  • Class attributes allow us to share a CSS rule to any element we choose by assigning the rule to a class attribute with a class selector. Then the class attribute can be assigned to the element in the HTML, enabling us to freely choose which elements will be assigned the CSS rule.

Git

  • We use Git in our local development environment to enable version control
  • git status: checks what branch we are currently on
  • git checkout -b branch-name: creates a new branch and switches to it
  • git add -A: adds all changes in the working directory to the staging area
  • A pull request is used to merge a comparison branch (eg the feature branch) into the base branch.
  • When using the push command, you can create a new remote branch or add new changes to an existing branch.

JavaScript

  • JavaScript is a high-level dynamic programming language that allows users to interact with websites
  • In JS we declare a variable using var, then = whatever data ; EG var shape = "circle";
  • A variable is a named container that allows us to store data in our code
  • Control flow is the order in which a computer executes code in a script
  • We can use the console.log method to output a message to the web console by adding an argument. EG console.log(topics)
  • With an if statement, code will only execute if the statement meets a condition, or is true. === checks if two values are equal
  • To create an array we use var name = [group of values] EG var shapes = ["triangle", "square", "pentagon", "circle"];
  • A for loop contains the starting point for our loop, the condition, then the iteration (EG X++)
